# ITH Hongsa — Overhaul Monitoring Dashboard

แดชบอร์ดติดตามความคืบหน้างาน **Overhaul มอเตอร์** ของสายพานลำเลียงและเครื่องจักร
(Progress Plan of Overhaul — Motor Line Conveyor & Machines) สำหรับ **Italian Thai Hongsa Co.,Ltd**

> ข้อมูล ณ วันที่ **20 มิถุนายน 2026** · ดึงจากชีตระบบ `SPD / BWE / BELT / CR / 2200 / 1800`
> ของไฟล์ `MASTER OH# STP 20.06.2026.xlsx`

## เปิดใช้งาน

เปิดไฟล์ [`Dashboard.html`](Dashboard.html) ด้วยเว็บเบราว์เซอร์ได้ทันที (ดับเบิลคลิก)
— เป็นไฟล์ **self-contained** ฝังข้อมูลไว้ในตัว ไม่ต้องต่ออินเทอร์เน็ตและไม่ต้องติดตั้งอะไรเพิ่ม

## ภายในแดชบอร์ด

แบ่งตามระบบเครื่องจักรเป็น **7 ปุ่ม (กดเข้าดูได้)** — `ทั้งหมด` + `SPD` `BWE` `CR` `BW2200` `BW1800` `BELT`

- **ปุ่มเลือกระบบ** — แต่ละปุ่มแสดงจำนวนมอเตอร์ + ไฟเตือนจำนวนที่เกินกำหนด OH
- **KPI การ์ด** (อัปเดตตามระบบที่เลือก) — จำนวนมอเตอร์, Running, เกินกำหนด, ใกล้ครบกำหนด, กำลังรวม (MW), อายุการใช้งานเฉลี่ย
- **กราฟสถานะ** (donut) · **แถบความเร่งด่วน OH#2** (Overdue / Due soon / OK) · **กราฟแยกตามระบบ/จุดติดตั้ง**
- **รายการมอเตอร์แบบกดกางดูได้** — คลิกที่มอเตอร์แต่ละตัวเพื่อดูข้อมูลครบ (SMU, ชั่วโมงคงเหลือ, **วันครบกำหนด OH ครั้งหน้า**, serial, ตำแหน่ง drive ฯลฯ)
- **ค้นหา + ตัวกรอง "เฉพาะเกินกำหนด"** ในแต่ละระบบ
